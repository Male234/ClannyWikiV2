# `c!pendingjoin`

Manage people pending for your group

:::info
This command requires the ClannyBot connected to your group to have "Manage pending" permissions on Roblox
:::

## Syntax

- `c!pendingjoin` View pending users in a pagination embed
- `c!pendingjoin accept [User]` Accept a pending user
- `c!pendingjoin decline [User]` Decline a user's join request
